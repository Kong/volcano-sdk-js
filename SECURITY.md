# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues,
pull requests, or discussions.

To report a vulnerability in Volcano SDK or other Kong software, email
vulnerability@konghq.com with the details. Include the affected package version,
reproduction steps, and any relevant impact assessment when possible.

For more information, see Kong's vulnerability disclosure program:
https://konghq.com/compliance/vuln-disclosure
